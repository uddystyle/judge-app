# Stripe 商品・価格の設定手順

このガイドに従って、Stripeダッシュボードで商品と価格を作成してください。

## 前提条件

- Stripeアカウントにログイン済み
- テストモードに切り替え済み（画面左上のトグルで確認）

## 手順

### 1. Stripeダッシュボードにアクセス

https://dashboard.stripe.com/ にアクセスし、**テストモード**に切り替えてください。

### 2. 商品ページに移動

左メニューから **「商品カタログ」** → **「商品」** をクリック

### 3. スタンダードプランの作成

#### 3.1 商品を追加

1. **「商品を追加」** ボタンをクリック
2. 以下の情報を入力:
   - **商品名**: `TENTO スタンダードプラン`
   - **説明**: `月間無制限セッション、100名まで/セッション、20名まで検定員、検定モード+大会モード+研修モード、スコアボード公開機能`

#### 3.2 価格を追加

**月額プラン:**

- **価格**: `980`
- **通貨**: `JPY`
- **請求期間**: `月次`
- **Price ID をメモ**: 作成後に表示される `price_xxxxx` をコピーしてメモ
  price_1SPHtjIsuW568CJsdqnUsm9d

**「別の価格を追加」** をクリック

**年額プラン:**

- **価格**: `9800`
- **通貨**: `JPY`
- **請求期間**: `年次`
- **Price ID をメモ**: 作成後に表示される `price_xxxxx` をコピーしてメモ
  price_1SPHurIsuW568CJsFfJ6kwYV

3. **「商品を保存」** をクリック

---

### 4. プロプランの作成

#### 4.1 商品を追加

1. **「商品を追加」** ボタンをクリック
2. 以下の情報を入力:
   - **商品名**: `TENTO プロプラン`
   - **説明**: `月間無制限セッション、選手数無制限、検定員無制限、全機能利用可能、データ保存期間無制限`

#### 4.2 価格を追加

**月額プラン:**

- **価格**: `2980`
- **通貨**: `JPY`
- **請求期間**: `月次`
- **Price ID をメモ**: 作成後に表示される `price_xxxxx` をコピーしてメモ
  price_1SPHvrIsuW568CJsBsRymAvZ

**「別の価格を追加」** をクリック

**年額プラン:**

- **価格**: `29800`
- **通貨**: `JPY`
- **請求期間**: `年次`
- **Price ID をメモ**: 作成後に表示される `price_xxxxx` をコピーしてメモ
  price_1SPHwCIsuW568CJsuuhrug0G

3. **「商品を保存」** をクリック

---

## メモした Price ID の整理

以下のフォーマットで Price ID を整理してください：

```
スタンダード月額: price_xxxxxxxxxxxxxxxxxxxxx
スタンダード年額: price_xxxxxxxxxxxxxxxxxxxxx
プロ月額: price_xxxxxxxxxxxxxxxxxxxxx
プロ年額: price_xxxxxxxxxxxxxxxxxxxxx
```

## 次のステップ

Price ID をメモしたら、これらを `.env` ファイルに追加します。
開発者にメモした Price ID を共有してください。

---

## 参考: 本番環境への移行時

本番環境でサービスを開始する際は：

1. Stripe ダッシュボードを **本番モード** に切り替え
2. 同じ手順で商品と価格を作成
3. 本番環境の Price ID を取得
4. Vercel の環境変数を本番用に更新

テストモードと本番モードで Price ID は異なりますので、必ず両方を管理してください。
